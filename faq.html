你可能正在問

我們把最常見、最關鍵的問題放在前面。每一則回答都以「可理解、可操作、可驗證」為原則。

費用與方案是怎麼計算？▾
目前為 Beta 邀請制，方案會依「功能開放範圍 / 使用量 / 區域」分級。加入試用名單不需付費；正式方案以官網公告與開通信為準。
建議：如果你只想先體驗互動與設定流程，先加入試用名單即可。
需要信用卡才能開始嗎？▾
加入試用名單不需要信用卡。若你之後選擇升級正式方案，才會依你選擇的支付方式進行付款設定。
隱私怎麼做？我不想留下敏感資料。▾
我們以「最小化蒐集」與「可控可刪除」為原則：你可以使用最少資料啟動、在設定中管理資料留存，並以產品頁的『我們的界線』作為承諾基準。
想快速了解？建議先看 我們的界線。
我可以隨時取消嗎？取消後會怎樣？▾
可以。一般做法是：你在帳號/訂閱設定一鍵取消，取消後仍可使用到當期結束；若有試用期，則在試用到期前取消就不會進入下一期計費（以你當時方案規則為準）。
支援哪些平台？需要下載 App 嗎？▾
目前以 Web App 為主，支援手機與電腦瀏覽器（Chrome / Safari / Edge）。不必先下載原生 App；你也可以把 Web App 加到主畫面當成捷徑使用。
我可以匿名使用嗎？▾
可以。你可以用最少資訊註冊（例如只用 Email），並在設定中管理可見資訊與通知方式。
為什麼會有『安全與界線』？這跟產品體驗有關嗎？▾
有關。產品體驗越沉浸，就越需要清楚界線來保護你：包含內容規範、資料控管、以及互動方式的透明度。界線不是限制，而是讓使用更安心。
遇到問題要找誰？▾
你可以先查本頁 FAQ；若仍需要協助，請到 聯絡我們，我們會依問題類型（客服/媒體/法務與隱私）導向對應窗口。

信任與界線（摘要）

我們把信任設計成產品的一部分，而不是事後補充條款。

  • 最小化蒐集：只拿需要的
  • 可控可刪除：你能管理留存
  • 透明規則：互動與內容規範清楚
